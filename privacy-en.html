4. Internet Connections and External APIs

üì° Internet Connection

SubsFly ONLY connects to the internet for the two specific features described below. There is no other external communication, tracking, analytics, or data transmission.

SubsFly makes occasional internet connections only for the following optional features:

üí±Real-Time Currency Conversion

  • What information is sent: Only the base currency code (e.g., EUR, USD, GBP)
  • Where it's sent: ExchangeRate-API (api.exchangerate.host)
  • URL used: https://api.exchangerate.host/latest?base=[CURRENCY]
  • What we receive: Updated exchange rates
  • Personal data shared: NONE. No information about your subscriptions, prices, amounts, or personal data is sent
  • When it happens: Only when the app needs to update rates (maximum once every 24 hours)
  • Storage: Rates are saved locally to work offline

üé®Platform Logos

  • What information is sent: Only the web domain of the platform (e.g., "netflix.com", "spotify.com")
  • Where it's sent: Google Favicon API (www.google.com)
  • URL used: https://www.google.com/[domain]
  • What we receive: The official logo image in PNG format
  • Personal data shared: NONE. Only the public domain name
  • When it happens: Only when viewing a subscription without a downloaded logo
  • Storage: The logo is saved locally and reused without downloading again
  • Offline functionality: Once downloaded, available without connection

üîí Privacy Guarantee:

  • These APIs DO NOT receive information about your subscription prices
  • DO NOT receive renewal dates or any personal data
  • DO NOT receive your device identifiers
  • We DO NOT track your searches
  • Connections are occasional and only when you initiate the action
